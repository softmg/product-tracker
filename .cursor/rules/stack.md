# Stack

## Backend
- Laravel 12 + PHP 8.3+
- PostgreSQL
- Docker / docker compose

## Frontend
- React 18+
- Effector 23+ (state management)

## Testing
- PHPUnit — backend unit/feature tests
- Vitest — frontend unit tests
- Playwright — E2E and UI tests

## Infrastructure
- Git hosting: GitLab
- Tracking: Jira
- User docs: Confluence
- Runtime: Docker / docker compose
